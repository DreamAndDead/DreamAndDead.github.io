Graphviz Examples
2021-03-22 14:48 folder [Tool]   label tool   graphviz   drawing   schedule 3 min 14 s.

usage

安装相应软件包

$ pacman -S graphviz xdot

新建 test.dot 文件，保存绘图代码。

使用 xdot 实时预览，

$ xdot test.dot

绘图结果，生成 png 图片，

$ dot -Tpng -o test.png test.dot

graph

undirected

graph {
   label="Undirected Graph";
   rankdir=LR;
   { rank=same; a, c, d };
   splines=line;

   a -- b -- c -- e -- a;
   a -- c -- d;
}

directed

digraph {
    a -> b;
    b -> c;
    c -> d;
    d -> a;
}

subgraph

asynomous

digraph G {
  main -> init -> make_string;
  main -> {parse, cleanup};
  main -> printf;
  parse -> execute -> {make_string, compare, printf};
}

named

Subgraph name must start with prefix cluster.

digraph {
    subgraph cluster_0 {
	label="Subgraph A";
	a -> b;
	b -> c;
	c -> d;
    }

    subgraph cluster_1 {
	label="Subgraph B";
	a -> f;
	f -> c;
    }
}

label

digraph {
    a -> b[label="0.2"];
    a -> c[label="0.4"];
    c -> b[label="0.6"];
    c -> e[label="0.6"];
    e -> e[label="0.1"];
    e -> b[label="0.7"];
}

straight line

graph {
    splines=line;

    subgraph cluster_0 {
	label="Subgraph A";
	a; b; c
    }

    subgraph cluster_1 {
	label="Subgraph B";
	d; e;
    }

    a -- e;
    a -- d;
    b -- d;
    b -- e;
    c -- d;
    c -- e;
}
