如何更好地度过一生？很多人都没有答案。
漫无目的，压抑本心，这种生活方式天生是分裂的，无法长久的。
曾有多时，感觉工作时如坐针毡；
几个年末，发现自己依旧在原地；
多次在噩梦中惊醒，惶恐一生会如此平凡下去。

#+begin_quote
其实，人生需要管理。
#+end_quote

管理，意味着设立目标，跟进进度，安排时间，调度资源等等。

现实中，很多人讨厌“管理”这个词。看到它就联想到规则，束缚，压制，是冰冷的。
的确，管理是多数时候是有些“不近人情”的，像一台无情的电脑，
给出一条一条指令，应该如何做。
而正是反人情，而能催促人去做正确的事。
因为“每个人都有惰性”，管理就是尝试去改善这一点。

多数人都有放长假的经验。
短短的周末两天，可以得到放松；
而一两个月的假期，失去时间观念，无聊，乏味，觉得时间难以消磨。

现代的工作制度，法定假期的安排，就应用了这一点。
在短暂的放松，不至于失去工作状态的连续性。

关于自我管理，前人已经总结了多种方法，个人感觉多少无法完全切合自身的需求。
从 todo 管理，时间管理，到精力管理，我个人是其爱好者和使用者。
如果以软件来类比，这些不同版本的软件都有些许 feature 没有包含。
todolist 缺少对长期连续性，让人沉浸在打勾的快感中；
时间管理过于关注效率，缺少对个人成长与长期目标的关注；
精力管理更进一步，注重高质量的时间，而没有质的改进。

本文的目的在于，尝试整合所有理论的优点，根据自身经验，
提出一种“更好”的自我管理方法。
  
* 常见陷阱

人生有很多陷阱。
我曾沉陷于某些陷阱，想必也陷住了其它很多人。

方向分散。
在我还是软件专业的大一新生时，总问过学长，
在这个专业的学习过程中遇到的最大的问题是什么？
他想了想说，“这个专业的东西太多了，
东西做着做着，一不小心就会偏到别的方向去。”
之前体会不深，现在深以为然。
这也是我看到 “T型人才” 理论时为何如此震惊。
“T型人才” 理论建议一个人在一个专业方向有深度，如 T 的竖线；
同时涉猎广泛，有其它专业的广度，如 T 的横线，
结合一些新奇的想法，同自己专业的前沿，探索出一些新东西。
若以这个理论评价自己，应该是 “.型人” 。
回想自己的学习生涯，总是抱着新奇的心态，
顺从内心贪婪的冲动，不断尝试新的东西，觉得总比继续深入挖掘更好，
结果没有一个方向的深入积累，人生在多个方向的浪费中涣散了。
公司的 CEO 对我说，“积累是重要的，要在某个方向有所积累”。
毕竟一个人时间有限，精力有限，不可能是全部，做到全部，
所有事情都做满，能在一个方向，一个点，已经很不错了。

时间的错觉。
时间是客观的，一小时永远是那么长，但是人的心理是变化的，
对时间有自己的感知。
就是说存在心理时间与实际时间，两种时间。
有的时候，觉得时间过得非常快，而有时度日如年，就是人的心理感知。
合理的时间观念，要逐渐顺应客观时间，是不可违背的，
而很多人如我有着非常不成熟的时间观念。
表现在，高估自己的效率，认为只需要很少的时间；
对时间抱有不切实际的幻想，认为之后一定会怎么怎么样。
从而行动拖延，消耗时间。

分裂的内心。
很多时候，会周期性地陷入迷茫的状态。
不知道自己在干什么，在做的事有什么意义，
未来会走到哪里，人生的价值在哪里。
这个时候只想逃离在做的事，
想放空自己，想静心思考，给自己一个答案。
其实每个人的内心都是分裂的，
如意识和潜意识。意识只能控制人体的一部分，
无法控制如呼吸，血液循环等潜在意识的范围。
两者各有自己的主张，如何协调两者的关系，
是人生中的大问题。

* 目的地

在现在的我看来，落入陷阱的最大原因在于，
太多人不知道，自己最终要到哪里，
只是一步一步乱走一通，
随波逐流，随遇而安。

每个人都应该在生命中尽早回答这样一个问题，
自己的终点在哪里？
如何定义终点，就是如何定义你人生的意义。
这是一种以终为始的思路，
当你遇到内心冲突，
反思现实困境和与梦想目标的偏差，
始终知道自己的方向在哪里。

每个人的终点是不同的。这一点要结合自己。
多数人盲目崇拜的成功可能不是你的终点，
环境与时遇可遇不可求。
切合自己的实际，毕竟，每个人都有自己的性格，兴趣，喜好，认知。
没有人的命运是相同的。

在个人的差异上，提出一个通用的假设，

#+begin_quote
人生的意义就是价值的最大化
#+end_quote

价值，就是所谓价值认同，价值观，中的价值二字。
每个人的价值观都不同，有人热衷财富，有人追求幸福感，
有人只想环游世界，对于不同的观点，我们应该尊重不同。

遵从价值的方向才是最为充实的方向。
比如听到人说，“这么做没意义，没有价值，简直是浪费时间。”
这时他就在谈论意义和价值。
只有合乎自身价值认同的，人才会感觉到充实，
因为自己所做的，和价值方向相同的。
如果不追求价值，平凡的度过一生，很难找到自身的意义，
更糟糕的是，逆向而行，去破坏去减少价值，内心势必是痛苦的。

* 理论

在上面的基础上，提出一种理论，
如何以终点为目标，做好自我管理。

** 定义
*** Agent & State

Agent 是一个广泛的概念，可以代表一个人，一个公司，或者一个国家，
取决于划分的层次。

从量化的角度看，更乐意如下定义，

#+begin_quote
Agent 是一个资产集合体
#+end_quote

资产可划分为身内之物和身外之物。
于个人而言，人生在世，身内之物是与生俱来的。包括自己的
- 身体，组成自身的物质基础
- 精神，自我意识所在
- 智力，做事能力，思考能力
- 情感，共情力，与他人的共鸣
而身外之物是逐渐积累起来，为自己所有，所用的，常见如
- 金钱
- 人际关系
- 权力
- 声望

每种资产都是可比较的。
如身体的强壮与瘦弱，精神的高尚与卑劣，
智力 IQ 的高低，财富的多寡，
权倾朝野与默默无名。

同时，每种资产都是可变的。
这也是人生游戏的魅力所在，
身体素质会在 996 中消退，也会在锻炼中更加强壮；
智力与技艺可在练习中逐渐提高，也会在网络游戏的沉沦中消退；
有人一夜暴富，有人顷刻破产；

#+begin_quote
State 是 Agent 的资产状态
#+end_quote
   
自我管理，就是管理好自己的资产。

*** Value

#+begin_quote
Value 是 Agent 对不同资产的认可度
#+end_quote

Value 即是所谓价值。

价值是主观意愿存在的。
每个个体都有自己的主观认同，
这因人而异。
也因如此，几乎没有普世的价值认同。
有人舍生取义，有人重利轻生。
甚至连和平都不是被普世认同的，
极端分子无时不在挑战这个美好愿景。

价值认同也可以分为内外两部分。
注重内在价值的人，崇尚内修，
磨砺心智，寻求精神的一；
注重外在价值的人，追逐现实，
积极创造。

我认为，价值认同不宜走极端，应内外兼修，
虽然每个个体的价值观是平等的，值得尊重的。
内修之人，容易陷入精神胜利法，
逃避现实，活在自己的精神胜利中；
外修的人，往往为名利遮蔽眼睛，
为外物所玩弄，失去真正的自我。

*** Object

#+begin_quote
Object 是 Agent 的目标 State
#+end_quote

Object 是依据 Agent 的价值认同而设定的目标状态。
根据价值最大化的原则，价值认同的更高的资产，
我们会希望获得更多。

目标是可以分解的。
个人目标往往是宏大的，遥远的，
而最终目标需要一步一步慢慢的来实现，
这就需要将目标分解
以指导人生规划。
从资产上看，目标可分解到不同资产层面；
从时间上看，长期目标可分解为短期目标。
这样一个个子目标串联起来，
共同组成最终的目标。

*** Thing

#+begin_quote
Thing 是两个 State 之间的连接
#+end_quote
   
Thing 是所谓“做事”中的事。
在当今分工合作的社会中，往往强调将事情做好。
事情可以将不同状态连接起来，做事可以改变 Agent 的状态。
假如终日什么都不做，必定停留在原地，一动不动。

状态改变的关键在于投入与回报。
事情本身有相应的投入与回报。
投入是做事的成本，消耗相应的资产；
事情达成之后，便会带来相应的资产回报。
两者之间的差值，造就了 Agent 资产状态的变化。
从当下到目标，有多个子状态相互连接。
Agent 通过做事，来逐渐改变状态，慢慢向目标靠近，达成目标。

上一节解释了核心部分的定义，后面解释
如何用一种行之有效的流程，将这些关键概念串联起来，
进行自我管理。

 1. Evaluate Self，评估自身，明确资产情况；认知自我，探讨自己的价值观
 2. Set Object，制定符合自身价值的长期目标

 6. Loop，自身状态已发生变化，重新开始这个循环，从重新评估开始

*** Evaluate Self

#+begin_quote
认识你自己
#+end_quote
   
评估自身，主要分为两部分，明确资产情况 和 探讨自己的价值观。

明确资产情况，就是明确你已拥有的。
正如定义中讲到的，资产分为多个类型，
建议按类型，各自进行评估。

探讨价值观就是明确自己当下的价值取向。
正如开篇提到的，很多人并不了解自己，往往在某个瞬间，
觉得失去了真我，行为和内心是相背离的。
那是因为从一开始没有认清楚自身，不断被外界推着走，
被忽略的内心迟早会来提醒你。
所以认知自己要什么，认为什么是有价值的，明确对各方资产的内心评价，
自己的方向才会更加明确。

关于如何分析自我，如何评估自己的价值取向，推荐两个方法：
1. 找一张空白的纸，分成六竖栏，写下自己喜欢和不喜欢的事；写下自己羡慕和讨厌的事；
   写下自己擅长与不擅长的事。相信在书写的过程中，会对自身有所启发
2. 尝试接触新事物。我们的圈子往往趋于稳定，接触了解的事物是有限的，
   而对事物的有限接触，不足以说明你就是什么样的人，没有其它的可能性。
   尝试接触新事物，可能会找到热爱。

更新：

什么是内心？
首先，内心不是心脏，而自己的大脑，自己的思想，自己的意识。
其次，每个人的内心都是分裂和冲突的。
因为内心有两部分，理性与感性，意识与潜意识。
如果意识之光是人类独一无二的天赋，
那潜意识的不受控制就是人类才独有的痛苦。

什么是真正的内心？
借用《象与骑象人》中的比喻，你认为自己是象，还是骑象人？
太多人（包括我）误以为自己是象，每次内心的冲突，
都是被内心的大象牵着走，都会向潜意识妥协。
“如果成为那样的人，就不是自己了”，
“正是因为自己是那样的人，才会是今天的自己”。
它拒绝改变，拼命维护自己的地位。
但是象是不会思考的，它不是理性。

“理性”的处境是非常艰难的。
意识的诞生不过是近千年来的事情，
而潜意识自生物存在那一刻就存在了。
当潜意识想要做什么，意识是没有力量与其直接抗衡的。

真正的内心应该以理性为主，“我思故我在”。
“我”应该是可主动的，可掌握的，而不是被盲目的冲动牵着走。
骑象人应该慢慢成长，慢慢教化大象潜在“蛮横”的部分，这正是人生的修行。

一个人是否要改变自己？
有人认为改变自己是不可能的，如古语“江山易改，本性难移”；
有人认为改变是无时无刻的，哲学上的辩题，一艘不断更换零件的船从什么时候开始不是自己？
事实上改变是必须的，现代竞争无处不在，几近是进化论的社会版，
只有最适应环境的改变才会存在。

重点是，内心中总有些疑惑存在，我是否要改变自己？
改变之后我还是我吗？我的朋友会怎么看待我？
我不想变成自己讨厌的人，朋友不认识的陌生人。
如果仔细推敲，这些问题都是站不住脚的，
而且都来自于内心的潜意味部分，
原因或者是因为恐惧，或者是因为拖延心理，
或者因为懒惰不想努力的借口。
所以，大胆向前走吧，接受过去，接受新事物，
人生有无限可能。
  
*** Set Object

#+Begin_quote
如果不知道去哪里，那么到达哪里也无所谓了。
#+end_quote

目标由价值而来。目标表示一个自我设立的状态，想要到达的状态，
而这个状态是由自我的价值观所引导的。
本着贪婪的原则，最看重的价值，在目标中占据的最多；
同时应该坚持平衡的原则，不应完全摒弃所有其它资产，
这样是坚持无法长远的。

目标分裂是痛苦的。现代人多数背负着外部的目标，
比如家庭的目标，公司的目标。
只有当外部目标与个人目标有高度重合的时候，才是契合的，
是最有效率的，否则目标的分裂会占据自己稀少的时间，
同时带来内心价值的缺失，这势必是痛苦的。

目标是一个探索过程。
目标宏大，遥远，对于当前的自己很难直接看到一条实现的路径。
因为当前的自己知识与眼界都有局限，
无法分析的非常长远。
所以设立目标固然重要，但如果目之所及，只有目标，就容易错过了其它的新事物，
而新事物往往对目标有所转机。
所以在自己的目标中，应该存在接受新消息，新事物的目标。

*** Plan Things

#+Begin_quote
始计
#+end_quote

计划主要做两件事，分解目标和安排事情。

分解目标，即将大目标分解为逐个力之所及的子目标。
可以在资产维度上进行分解，也可以在时间维度上进行分解。

安排事情，就是找到子目标间，成本最小的做事路径，
并安排时间作为 todo 事项。
每件事都对应着资产成本，如身体，金钱，人脉，这就是成本。
实现一个子目标，有多种做事的途径来达成，
最优的计划就是找到成本最小的做事方式。

*** Execute Things

#+Begin_quote
再天才的计划也需要执行才能实现
#+end_quote

目标通过计划分解为事项，事项再安排为 todo。
执行就是根据时间表，调动相应精力和资产去处理。

执行需要人的自律，坚持，不拖延，
而且坚持一定的做事原则是必要的。
- 能早即早，事情要提前尝试，对于意外的变故才有缓冲的余地
- 保持好精力，注意劳逸结合
- 少做计划之外的事，很多时间无法按捺即刻的冲动，
  殊不知时间都在分毫间消失了

*** Refresh Self

#+Begin_quote
Version 1.0 -> 2.0 -> 3.0
#+end_quote

上面的理论规划的看似完美，其实有一个致命的局限性，
那就是没有考虑到个人的成长性。

为什么考虑成长性是必要的？
在向目标前进的过程中，自己不断经历新的事情，
智力会成长，精神会变化。
个人总体在向上提升，而价值观，目标，计划相比之下就是“旧版本”的。
最初可能年轻气盛，价值观并不成熟，偏激且凌厉，
从而制定不合理的目标。
而且由于眼界的限制，看不清到达目标的路，
制定的计划也不完美。
对于明显可达的目标，计划是明确有效的；
对于不明显的目标，迷雾重重，只能根据设想，去猜测一条路径，
其中少不了试错的过程，慢慢看清眼前的迷雾，扩大自己的认知范围，
这些都是宝贵的经验。

所以在过程中，要不断回头审视，总结经验，更新自己。

*** Loop

#+begin_quote
I'm a wired loop
#+end_quote

当更新结束之后，就要回头，以新的面貌，
重新开始这个过程。

整个循环中，周期是关键的。
推荐
- 一天，反思自己的周度目标
