# 安装 ide support for ue

在 vs installer 中，安装 ide support for ue

为 vs 提供如下功能
1. 在 vs 中查看 bp 引用
2. 可以添加 ue 类
3. 合并 log 到 vs
4. 查看一些 ue macro 的 expand

总体而言，作用不大

# 在 ue 中安装 vs integration tool plugin

[ms 提供的 ue 插件](https://github.com/microsoft/vc-ue-extensions)

支持 vs 中查看 bp 类
