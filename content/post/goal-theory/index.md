## key concept {#key-concept}

是一个有精神意识的资源集合体

身内之物

### value {#value}

对各项资源的重要程度的评价，由 agent 的特质决定


### object {#object}

目标是 agent 的一个状态点

目标是在某个时间点上的

可以根据资源来划分，也可以根据身份来划分

目标可以进行分解，小到力之所及

目标是具体的，可量化的


### result {#result}

实际结果

在某个时间点上的实际状态

可以和对应的目标对比，比较差距


### thing {#thing}

事情是改变 agent 状态的链接

通过做事，来达到目标

每件事都有自己的成本，会消耗一定的资源


## theory {#theory}

人生的目的地就是追求价值的最大化。

这个理论融合了以下方面的要点

-   OKR
-   投资
-   RL
-   高效能人士的习惯

{{< figure src="images/theory-loop.png" >}}

1.  evaluate，评估自身，探讨自己的价值观
2.  set object，制定最大化自身价值的目标
3.  plan，目标分解，安排事情
4.  execute，执行计划
5.  refresh，收集结果，总结经验
6.  loop，agent 自身状态已经发生变化，需要再次开始这个过程


### evaluate {#evaluate}

评估自我的资源状态，自我的价值取向

单纯内修的人，是一种逃避现实的精神胜利
单纯外修的人，往往失去真正的自我

分析自我

-   喜欢和不喜欢的事
-   羡慕和讨厌的事
-   擅长与不擅长的事

post
如何分析自我


### set object {#set-object}

目标表示一个时间点的状态

目标由价值而来
目标由现实的困境而来

内外兼修，平衡的原则

在生活中，不仅有自己的目标，还要实现外部的目标，比如在组织中。
只有外部目标与自己目标有高度重合，才是最有效率的人生，
否则目标的冲突会分割自己稀少的时间。

目标固然很重要，但如果目之所及，只有目标，就错过了其它的新事物。

所以在自己的目标中，应该添加关于接收新事物，新消息的目标，
打开接收消息，主动更新的窗口。

更新的速度
时刻影响 agent 自身
对于价值的判断，目标的设定与修改


### plan {#plan}

计划用来

-   分解目标，将大目标分解为力之所及的小目标
-   安排事情，针对某个目标，寻找出成本最小的做事路径，安排时间作为 todo

在组织中的目标分解，是将目标分解到同一时间的不同人身上，
而个人的目标分解，是将目标分解到自己一生的不同时间上。

每件事，都会消耗资源，如时间，金钱，能量，人脉，这就是成本部分。
如果有人无偿帮忙，是更好，或者资源交换，雇用，这便是做事的方法。

要实现一个目标，有无数种做事的组合可达成；
最优的计划就是成本最小的做事方式。

计划就是要合理的安排资源。

分解的小目标必须和大目标一样，是可量化的。
比如将 lua book 目标分解到最小，“发布 gc 章节”，看起来像是一件事情，
也可以看作是一个 bool 属性的小目标，因为最终只能衡量 发布/未发布 两种结果。


### execute {#execute}

执行考验人的自律，坚持，不拖延，不松懈

再天才的计划也需要执行才能实现。

目标计划的事情变为 todo 之后，根据安排，使用相应的精力和资源去处理，
就像是一个投资的过程。

执行是调度资源的能力。

做事原则

能早就不晚，事情要提前尝试，这样对于意外的变故，有缓冲的余地。

管理好精力，高质量的时间才是你需要的

少做目标之外的事，是对资源的浪费

时间都在分毫无关的时候，消失了
 即刻的冲动
 自律

行动才有了一切


### refresh {#refresh}

更新自我，对过去有良好的统计，整理和总结，使自己得到提升。

在前进的过程中，自己会不断更新，智力会成长，精神会变化，
所以要不断的回头审视，总结经验，提升自己。

对于可明显到达的目标，计划是有效的；
对于不明显的目标，迷雾重重，只能根据设想，去猜测一条路径，
其中少不了试错的过程，当看清眼前迷雾的时候，就扩大了自己的认知范围，
这时可以再次规划目标，计划路径。


### loop {#loop}

从 evaluate 重新开始

整个 loop 的周期可以为

一周，反思自己的月度目标   4

一月，反思自己的季度目标   3

季度，反思年度目标         4

年度，反思 5 年目标        5

5 年，反思人生目标         5
