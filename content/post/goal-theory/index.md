如何更好地度过一生？很多人都没有答案。
曾有多时，感觉工作如坐针毡；
多少年末，发现自己依旧在原地；
屡次被噩梦惊醒，惶恐会一直平凡下去。
压抑本心的生活，漫无目的的生活，都是无法长久的。

> 人生需要管理

管理可能是一个答案。
管理，意味着设立目标，跟进进度，安排时间，调度资源等等。

现实中，很多人讨厌“管理”这个词。
它使人联想到规则，束缚，压制，冰冷。
的确，有些管理方式是不近人情的，
只给出一条一条指令，指示如何如何去做。
反过来想，正是因为反人情，才更能催促人去做正确的事，
因为每个人都有惰性。
拿假期来说，
周末两天，可以在工作之余得到放松；
而长达两个月的寒暑假，人会在其中失去时间观念，觉得时间难以消磨，无聊，乏味。

关于自我管理，前人已经总结了多种方法，自己从实践中也受益匪浅。
从 todo 管理，时间管理，到精力管理，都和自身需求差一些契合。
todolist 缺少长期连续性，使人沉浸在打勾的快感中；
时间管理过于关注效率，缺少对个人成长与长期目标的关注；
精力管理注重高质量的时间，而没有质的改变。

本文的目的在于，
尝试根据自身经验，结合多个理论的优点，
设计一种“改进”的自我管理方法。


## 常见陷阱 {#常见陷阱}

人生有很多陷阱。
我曾沉陷于某些陷阱无法自拔，想必它们也困住了很多人。

**方向分散。**
当我还是软件专业的大一新生时，曾问过学长，
“在你的学习过程中遇到的最大问题是什么？”
他想了想说，
“这个专业涉及的东西太多了，
一不小心就会偏到别的方向去。”
之前无法体会，现在深以为然。
回想自己的学习生涯，总是抱着过于新奇的心态，
顺从内心贪婪的冲动，不断尝试新的东西，
觉得广泛涉猎比深入挖掘更具“充实感”，
最终没有在一个领域深入积累，精力在多个方向的挥霍中涣散了。
公司的 CEO 对我说，“积累是重要的，要在某个方向有所积累”。
一个人时间有限，精力有限，不可能做到全部，
能在一个方向一个点，将事情做满，已经很不错了。
这也是我看到 “T型人才” 理论时震惊而醒悟的原因。
“T型人才” 理论提倡人在一个专业方向有深度，如 T 的竖线；
同时涉猎广泛，有其它专业的广度，如 T 的横线，
专业的前沿结合新奇的想法，可以探索出一些新东西。
若以这个理论评价自己，应该是 “. 型人”，自觉羞愧。

**时间错觉。**
对时间的认知有两种，客观时间与心理时间。
时间是客观的，一小时长度永远不变；
但是人的心理是在变化的，对时间往往有自己的主观感知。
有的觉得时间过得飞快，而有时度日如年，
这就是人的心理在作祟。
合理的时间观念，应该逐渐顺应客观时间，
而很多人（如我）有着非常不成熟的时间观念。
高估自己的效率，以为只需要很少的时间；
认为几年之后一定可以做成某件事，以为自己的时间充裕。
当对时间抱有不切实际的幻想时，往往行动拖延，
而没有认识到，白白浪费今天的一小时，
和浪费明天的一小时是一样的。

**内心分裂。**
很多时候会陷入迷茫的状态，不停的自我追问，
自己到底在做什么？在做的事有什么意义？
未来会走到哪里？人生的价值在哪里？
这个时候只想逃离工作，逃离生活，
想放空自我，想静心思考，给自己一个答案。
其实每个人的内心都是分裂的，
自我感知到的是意识，还有绝大部分的潜意识无法感知。
意识只能控制人体的一部分，
无法控制如肠胃蠕动，血液循环等潜在意识的范围。
当两者各有自己的主张，
如何协调两者的关系，就成了人生中的大问题。


## 意义 {#意义}

落入陷阱的最大原因在于，
太多人不知道自己最终要到哪里，
只是随波逐流，随遇而安。

每个人都应该尽早回答这个问题，
自己的终点在哪里？
终点就是你人生的意义。
这是一种以终为始的思路，
当你内心迷茫时，
反思现实与梦想的偏差，
就知道自己的方向在哪里。

这里提出一个假设，

> 人生的意义就是价值的最大化

价值，就是所谓价值认同，价值观。
每个人的价值观都不同，有人热衷财富，有人追求幸福感，
有人钟爱环游世界。

遵从价值的方向才是最为充实的方向。
有时听到人说，“这么做没意义，没有价值，简直是浪费时间。”
这时他就在谈论意义和价值。
如果不追求价值，平凡的度过一生，很难找到自身的意义。
糟糕的是，有人逆向而行，去破坏价值，其内心势必是痛苦的。


## 框架 {#框架}

在踩过诸多陷阱之后，尝试设计一个框架，如何避开它们。

核心是，以终点为目标，做好自我管理。


### 定义 {#定义}


#### Agent & State {#agent-and-state}

Agent 是一个抽象的概念，用于代表一个有精神的个体。
从量化的角度，提出如下定义，

> Agent 是一个资产集合体

资产可划分为身内之物和身外之物。

人生在世，身内之物是与生俱来的，包括

-   身体，自身的物质基础
-   精神，自我意识
-   智力，做事能力，思考能力
-   情感，共情力

身外之物是逐渐积累起来的，为自己所有，所用的，比如

-   金钱
-   人际关系
-   权力
-   声望
-   等

每种资产都是可比较的。
身体的强壮与瘦弱，精神的高尚与卑劣，
智力 IQ 的高低，财富的多寡，
权倾朝野与默默无名。

同时，每种资产都是可变的，这也是人生游戏的魅力所在。
身体素质会在 996 中消退，也会在锻炼中更加强壮；
智力与技艺可在练习中逐渐提高，也会在网络游戏的沉沦中消退；
有人一夜暴富，有人顷刻破产。

> State 是 Agent 的资产状态

自我管理，就是管理好自己的资产。


#### Value {#value}

> Value 是 Agent 对不同资产的认可度

Value 即是价值认同，价值观。

价值观是一种主观意愿。
每个个体都有自己的主观认同，因人而异。
正因如此，没有普世的价值认同。
有人舍生取义，有人重利轻生，
甚至连和平都不是被所有人认同的。

价值认同也可以分为内外两部分。
注重内在价值的人，崇尚内修，磨砺心智，寻求精神的一；
注重外在价值的人，追逐现实，积极创造。
私认为，价值认同不宜走极端，而应内外兼修。
内修之人，容易陷入精神胜利法，逃避现实，活在自己的精神胜利中；
外修的人，往往为名利遮蔽眼睛，为外物所玩弄，失去真正的自我。


#### Object {#object}

> Object 是 Agent 的目标 State

不同的价值观，对应不同的目标状态。
根据价值最大化的假设和贪婪原则，
价值认同的更高的资产，
Agent 会希望获得更多。

目标往往是宏大遥远的，
常常需要被分解为子目标，一步一步来实现。
从资产上看，目标可分解到不同资产层面；
从时间上看，长期目标可分解为短期目标。
一个个子目标串联起来，共同组成最终目标。


#### Thing {#thing}

> Thing 是两个 State 之间的连接

Thing 是所谓“做事”中的事。
事情将不同状态连接起来，做事可以改变 Agent 的状态。
在分工合作的社会中，往往强调将事情做好，
假如终日什么都不做，必定停留在原地，不会发生改变。

改变状态的关键在于投入与回报。
事情本身对应投入与回报。
投入是做事的成本，消耗相应的资产，
达成之后，便会带来其它资产回报，
两者间的差值，造就了资产状态的变化。
从当下到最终目标，有多个子状态相互连接，
Agent 通过做事，逐渐改变状态，慢慢向目标靠近。


### 流程 {#流程}

行之有效的流程，是自我管理的关键。

{{< figure src="images/theory-loop.png" >}}

1.  Evaluate Self，评估自身，明确资产情况；认知自我，探讨价值观
2.  Set Object，制定最终目标
3.  Plan Things，目标分解，分解为子目标；安排事情
4.  Execute Things，执行计划，做事
5.  Refresh Self，更新状态，总结经验
6.  Loop，自身状态已发生变化，重新开始循环


#### Evaluate Self {#evaluate-self}

> 认识你自己

评估自身，主要分为两部分，明确资产情况 和 探讨价值观。

明确资产情况，就是明确已拥有的。
正如定义中讲到的，资产分为多个类型，
只需按类型，各自进行评估即可。

探讨价值观就是明确自身的价值取向。
正如开篇提到的，很多人并不了解自己。
往往在某个瞬间，觉得行为和内心是相背离的。
因为一开始没有认清自身，不断被外界推着走，
被忽略的内心迟早会来提醒你。
认知自己要什么，认为什么是有价值的，明确对各种资产的内心评价，
自己的知行才会统一。

关于如何评估自身的价值取向，推荐两个方法：

1.  找一张空白的纸，分成六竖栏：
    分别写下自己
    羡慕和讨厌的人；
    喜欢和不喜欢的事；
    擅长和不擅长的事。相信在书写的过程中，会对自身有所启发。
2.  尝试接触新事物。我们的圈子往往趋于稳定，接触了解的事物是有限的，
    对事物的有限接触，不足以说明你就是这样的人，没有其它的可能性。
    尝试接触新事物，可能会找到热爱。

**什么是真正的内心？**

首先，内心不是心脏，而自己的大脑，自己的思想，自己的意识。
其次，每个人的内心都是分裂和冲突的。
因为内心有两部分，理性与感性，意识与潜意识，
意识之光是人类独一无二的天赋，
而潜意识的不受控制是人类才独有的痛苦。

借用《象与骑象人》中的比喻，你认为自己是象，还是骑象人？
太多人（包括我）误以为自己是象，每次内心的冲突，
都被内心的大象牵着走，都向潜意识妥协。
“如果成为那样的人，就不是自己了”，
“正是因为自己是那样的人，才会是今天的自己”。
它拒绝改变，拼命维护自己的地位。
但是象是不会思考的，它不是理性。

“理性”的处境是艰难的。
意识的诞生不过是近千年来的事情，
而潜意识自生物存在那一刻就存在了。
当潜意识想要做什么，意识是没有力量与其直接抗衡的。

真正的内心应该以理性为主，“我思故我在”。
“我”应该是主动的，而不是被盲目的冲动牵着走。
骑象人应该慢慢成长，慢慢教化大象潜在“蛮横”的部分，
这正是人生的修行。

**一个人是否要改变自己？**

有人认为改变自己是不可能的，如古语“江山易改，本性难移”；
有人认为改变是无时无刻的，哲学上的辩题，一艘不断更换零件的船从什么时候开始不是自己？

重点是，内心中总有疑惑，是否要改变自己？
改变之后我还是我吗？其它人会怎么看待我？
我不想变成自己讨厌的人，大家不认识的陌生人。
如果仔细推敲，这些问题都是站不住脚的，
而且都来自于内心的潜意识，
原因或是恐惧，或是懒惰不想努力。

事实上改变是必须的，现代竞争无处不在，
只有最适应环境的改变才会存在。
所以，大胆向前走吧，接受过去，接受新事物，
接受自己的其它可能。


#### Set Object {#set-object}

> 如果不知道去哪里，那么到达哪里也无所谓了。

目标由价值而来。目标表示一个自我设立的状态，想要到达的状态，
而这个状态是由自我的价值观所引导的。
本着贪婪的原则，最看重的价值，在目标中占据的最多；
同时应该坚持平衡的原则，不应完全摒弃所有其它资产，
这样是坚持无法长远的。

目标分裂是痛苦的。现代人多数背负着外部的目标，
比如家庭的目标，公司的目标。
只有当外部目标与个人目标有高度重合的时候，才是契合的，
是最有效率的，否则目标的分裂会占据自己稀少的时间，
同时带来内心价值的缺失，这势必是痛苦的。

目标是一个探索过程。
目标宏大，遥远，对于当前的自己很难直接看到一条实现的路径。
因为当前的自己知识与眼界都有局限，
无法分析的非常长远。
所以设立目标固然重要，但如果目之所及，只有目标，就容易错过了其它的新事物，
而新事物往往对目标有所转机。
所以在自己的目标中，应该存在接受新消息，新事物的目标。


#### Plan Things {#plan-things}

> 始计

计划主要做两件事，分解目标和安排事情。

分解目标，就是将大目标分解为逐个力之所及的子目标。
可以在资产维度上进行分解，也可以在时间维度上进行分解。

安排事情，就是找到子目标之间成本最小的做事路径，
并安排时间作为 todo 事项。
每件事的消耗都不同，如身体，金钱，人脉，这就是成本。
实现一个子目标，存在多种做事的途径来达成，
最优的计划就是找到成本最小的做事方式。

#### Execute Things {#execute-things}

> 再天才的计划也需要执行才能实现

执行就是根据时间表，调动相应精力和资产去处理 todo 事项。

执行需要人坚持，自律，不拖延，
保持一定的做事原则是必要的。

-   要事第一，根据未来折扣理论，即使未来的事情更重要，人还是容易为眼前要做的事所迷惑
-   能早即早，事情要提前尝试，对于意外的变故才有缓冲的余地
-   保持精力，注意劳逸结合
-   少做计划之外的事，放纵即刻的冲动，时间都在分毫间消散


#### Refresh Self {#refresh-self}

> Version 1.0 -> 2.0 -> 3.0

以上理论看似完美，其实有一点致命的局限，
没有考虑到个人的成长性。

**为什么考虑成长性是必要的？**
在向目标前进的过程中，自身不断产生新的经历，
智力会成长，精神会变化，
总体在向上提升，而价值观，目标，计划相比之下已经是“旧版本”。
初期年轻气盛，价值观并不成熟，偏激且凌厉，
制定的目标也不合理。
由于眼界的限制，看不清到达目标的路，计划也不完美。
对于明显可达的目标，计划是明确有效的；
对于不明显的目标，迷雾重重，只能根据设想，去猜测一条路径，
其中少不了试错的过程，只能慢慢看清眼前迷雾，扩大认知范围，
增加经验。

所以整个过程中，要考虑自身的成长性，
不断回头审视，总结经验，更新自己。


#### Loop {#loop}

> I'm a wired loop

反思更新之后，就要回头以新的面貌，
重新开始整个过程。

循环周期是关键的。
推荐

-   一天，反思自己的周度目标
-   一周，反思自己的月度目标
-   一月，反思自己的季度目标
-   季度，反思年度目标
-   年度，反思 5 年目标
-   5 年，反思人生目标

发现偏差时，以“新的自我”去积极修正。


## 引用 {#引用}

这一切都开始于阅读《奇特的一生》，
内心震撼之余，后悔自己浪费了太多时间，
从而开始接触时间管理。

《高效能人士的七个习惯》总结了多个经典原则，
其以终为始，不断更新是过程中非常重要的部分。

《拖延心理学》是一本杰作，从中看到太多自己在执行方面的漏洞。

《象与骑象人》更新了自己对于内心的认知，
原来内心的分裂的冲突是必然的，
自己之前都在错误地维护象的地位。

强化学习是一个精致的理论。
理论设定一个智能体，通过和环境交互，不断从环境中得到反馈，
来进行学习。同时提出 Exploration 与 Exploitation 的矛盾。
这个理论对自己的学习方式带来很大的影响。
“Agent” 即向其致敬。

Object 取自 OKR 理论的 O，无论是企业还是个人，Object 都是通用的。

计划事项安排为 todo 来执行取自 GTD 理论。
事情的投入与回报则是投资理论的基础。
